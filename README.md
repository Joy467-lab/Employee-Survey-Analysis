# Employee Satisfaction Survey Analysis Dashboard

## Project Overview

This project analyzes an Employee Satisfaction Survey dataset using Microsoft Excel and Power Query. The aim was to clean, transform, analyze, and visualize employee survey responses to identify trends in employee satisfaction across departments, employee levels, and survey questions. The final deliverable is an interactive Excel dashboard that provides insights to support data-driven decision-making and improve employee engagement.

---

## Project Objectives

The objectives of this project were to:

* Clean and prepare the raw survey dataset using Power Query.
* Analyze employee satisfaction levels.
* Compare satisfaction across departments.
* Compare satisfaction across different employee levels.
* Identify the highest and lowest rated survey questions.
* Visualize findings through an interactive dashboard.
* Generate actionable recommendations based on the analysis.

---

## Dataset Description

The dataset consists of employee survey responses collected across multiple departments within the organization.

The dataset includes information such as:

* Department
* Survey Questions
* Response Score
* Response Text
* Employee Level
* Survey Completion Status

Only completed survey responses were included in the final analysis to ensure accurate reporting.

---

## Data Cleaning Process

Data cleaning was carried out using Microsoft Power Query before the data was loaded back into Excel for analysis.

The following cleaning steps were performed:

* Imported the raw dataset into Power Query.
* Filtered the dataset to include only completed survey responses.
* Removed unnecessary columns.
* Checked and corrected data types.
* Cleaned the survey question text by removing numbering from the beginning of each question.
* Created an Employee Level column for easier analysis.
* Created a Response Category column to classify responses as Positive, Neutral, or Negative.
* Loaded the cleaned dataset back into Excel.

---

## Data Analysis

After cleaning the dataset, Pivot Tables were created to summarize the survey data.

The analysis included:

* Average Satisfaction by Survey Question
* Average Satisfaction by Department
* Average Satisfaction by Employee Level
* Top 5 Highest Rated Survey Questions
* Top 5 Lowest Rated Survey Questions
* Distribution of Response Categories

These Pivot Tables served as the data source for the dashboard visualizations.

---

## Dashboard

An interactive dashboard was created in Microsoft Excel to present the findings in a clear and user-friendly format.

The dashboard includes the following Key Performance Indicators (KPIs):

* Average Satisfaction Score
* Total Completed Responses
* Overall Satisfaction Percentage

The dashboard also includes the following visualizations:

* Average Satisfaction by Department
* Average Satisfaction by Employee Level
* Top 5 Highest Rated Survey Questions
* Top 5 Lowest Rated Survey Questions
* Response Category Distribution

---

## Key Insights

The analysis revealed several important findings:

* Overall employee satisfaction was moderate across the organization.
* Satisfaction levels varied across departments.
* Employee satisfaction differed across employee levels.
* Some survey questions consistently received higher ratings, indicating organizational strengths.
* Other survey questions received lower ratings, highlighting areas that require improvement.
* The distribution of positive, neutral, and negative responses provided an overall view of employee sentiment.

---

## Recommendations

Based on the analysis, the following recommendations are proposed:

* Improve communication between management and employees.
* Strengthen employee recognition and reward programs.
* Address the concerns reflected in the lowest-rated survey questions.
* Continue monitoring employee satisfaction through regular surveys.
* Encourage departments with lower satisfaction scores to implement targeted improvement initiatives.
